# Book Store


#### Created with:

-   Typescript
-   ReactJS
-   Redux
-   Bootstrap
-   express.js
-   mongoose

#### Tools and libs:

-   dotenv
-   nodemon (autorestart server)
-   concurrently
-   bcryptjs
-   jsonwebtoken
-   morgan (logger)
-   multer (uploading images)


#### Dev:

NODE_ENV=...
MONGO_URI=...
JWT_SECRET=...
```

1. Run `npm i` in the root folder. Repeat it in the frontend folder
2. Run `npm start` in the root (or `npm run server` and `npm run client` separately)
